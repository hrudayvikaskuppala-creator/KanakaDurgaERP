# Splash Screen Assets

Drop your own files here to personalize the loading screen that
appears right after login:

| File                 | Purpose                                   |
|----------------------|--------------------------------------------|
| `splash_image.png` or `splash_image.jpg` | Image shown on the loading screen (e.g. a devotional image of your choice, or your shop logo). Recommended size: roughly 400x400px or larger, square or portrait. |
| `splash_audio.mp3`   | Audio played once while the loading screen is shown (e.g. Govinda Namalu or any devotional/branding audio track you have the rights to use). |

If either file is missing, the app automatically falls back to a
clean branded loading animation instead — it will never crash if
these files aren't present.

**Note on copyright:** please only add image/audio files that you
own or are licensed to use in your business software. Add the files
here, keep the exact filenames above, and the app will pick them up
automatically the next time it runs.
